## 👋 About Me

I'm a Senior Software Engineer based in **Irving, TX** with experience across the full stack — from low-latency Java microservices to React frontends to Python ML pipelines. I've worked across fintech, data platforms, and enterprise migrations.

Most recently, I've been deep in the **Capital One ↔ Discover merger** — one of the largest bank mergers in US history — handling backend service migration, data reconciliation, and API modernization across teams. It's messy, complex, and exactly the kind of thing I love.

Outside of work I've been building and exploring **LLM-powered applications**, plugging into APIs like Gemini and OpenAI, and figuring out where GenAI actually makes systems better vs. just noisier.

## 🤖 AI / LLM Work

I've been spending a lot of personal time exploring what's actually possible with today's LLMs — not just prompt engineering but real integrations:

- Built a **resume + JD analyzer** using Gemini API (free tier) — extracts skills, gaps, and gives structured match scores
- - Experimenting with **RAG pipelines** — chunking, embedding, retrieval using FAISS and Chroma
  - - Explored **fine-tuning smaller models** (DistilBERT, Mistral) for domain-specific classification tasks
    - - Integrated LangChain with existing REST services to add **conversational context** to internal tools
      - - Keeping a close eye on **agentic frameworks** — AutoGen, CrewAI, LangGraph — for where this is all going
       
        - The intersection of large-scale distributed systems and AI is where I want to be for the next decade.
